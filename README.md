# Nirmal Premil - Portfolio

A personal portfolio website showcasing my work as a Software Engineer with 3.3 years of experience in frontend development, e-commerce frameworks, and CMS-driven architectures.

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **UI:** React 19
- **Data Layer:** Apollo Client 4 + GraphQL
- **Styling:** Global CSS with CSS custom properties for theming
- **Fonts:** Google Fonts (Poppins, Raleway), Monoton (custom)

## Features

- Single-page layout with 5 sections: Home, Skills, Portfolio, About, and Contact
- Dynamic color theme switcher with 6 preset accent colors
- Animated section transitions with breaker overlay effect
- Responsive design (desktop, tablet, mobile)
- Hamburger menu with full-screen overlay navigation
- Contact form with client-side handling
- Downloadable resume

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the portfolio.

## Scripts

| Command         | Description                              |
| --------------- | ---------------------------------------- |
| `npm run dev`   | Start the development server             |
| `npm run build` | Create a production build (lint + types) |
| `npm run start` | Serve the production build               |
| `npm run lint`  | Run ESLint                               |
